# xkcd-dl
Download all xkcd comics

## Installation
* Clone the [Git Repository](https://github.com/ninenine/xkcd-dl) and edit with your favorite editor. e.g. Visual Studio, Visual Studio Code
* Launch with `F5` or `Ctrl+F5` (The usual way)
